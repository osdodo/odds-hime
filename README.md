## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Configuration

Get your API key from [console.machinefi.com](https://console.machinefi.com/)

Create `.env.local`:
```bash
NEXT_PUBLIC_TRIO_BASE_URL=https://console.machinefi.com
```

## Usage

1. Enter Stream URL (e.g., YouTube live)
2. Enter API Key
3. Click "Start Analysis"
